# Signal Forge v2
**Multi-agent AI crypto trading system**

Paper trading system using Alpaca with DeepSeek R1 via Ollama for multi-agent signal filtering and programmatic execution.

## What It Does
- Multi-agent signal filtering across crypto markets
- DeepSeek R1 inference via Ollama for local LLM processing
- Paper trading on Alpaca for strategy validation
- Multi-exchange research automation

## Stack
Python · Alpaca API · Ollama · DeepSeek R1
